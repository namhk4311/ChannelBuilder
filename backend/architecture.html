VNG Insider · TikTok Agent SYSTEM DESIGN

5 agents + orchestrator · localhost dashboard · vòng học khép kín (vòng sau giỏi hơn vòng trước)
Pipeline (A → B → C → D → E) Self-learning loop (E → B) Human-in-the-loop Orchestration / storage

Architecture diagram of the VNG Insider TikTok Agent: five agents coordinated by an orchestrator, controlled by a localhost dashboard, with storage and external services.

ORIGINAL PRINCIPLE
AI execute + learn · Human decide. Agent chỉ đề xuất nhân rộng, người bấm xác nhận. Không bao giờ tự động scale.
ABSOLUTE GATE
Hai phanh tránh "best of a bad batch". Top 20% lô và vượt ngưỡng tuyệt đối — cả lô lỗ thì không scale cái "đỡ lỗ nhất".
2 NGUỒN HỌC
Trend ngoài (A · tuần) + insight nội bộ (E · ngày) đều đổ về Creative Brain — vòng sau giỏi hơn vòng trước.
FALLBACK
TikTok API duyệt lâu → Publisher xuất file + caption, human bấm đăng tay (vẫn đúng "Human decide").
